<objective>
Create browser-based wizard UI with bilingual support and real-time SSE deployment narration streaming.

Purpose: Complete MCP-02 (live demo with side-by-side layout) and MCP-03 (basic UI for narration) by delivering an accessible, bilingual wizard interface that displays deployment progress in real-time.

**Important architecture note:** This UI shows NARRATION of what is being deployed. The actual deployment is triggered EXTERNALLY by the user running `/stagegold` via Claude Code in VS Code Pro. The web UI and Claude Code run in parallel - the UI provides visual feedback while Claude Code executes the actual MCP deployment to Power BI Desktop.

**Split screen approach:** Based on RESEARCH.md findings, local Power BI Desktop cannot be embedded in an iframe. The recommended approach is **side-by-side window arrangement**: user manually positions browser (JobForge UI) and Power BI Desktop windows on screen. The UI will include instructions for this setup.

Output: Fully functional web UI with 4-step wizard (Load, Power BI Prompts, Review, Catalogue), real-time deployment narration, EN/FR toggle, and WCAG 2.1 AA compliance.
</objective>

2. Create `src/jobforge/demo/static/js/sse.js`:
   - DeploymentStream class:
     ```javascript
     class DeploymentStream {
       constructor(url, options = {}) {
         this.url = url;
         this.onStart = options.onStart || (() => {});
         this.onTable = options.onTable || (() => {});
         this.onRelationship = options.onRelationship || (() => {});
         this.onComplete = options.onComplete || (() => {});
         this.onError = options.onError || (() => {});
         this.source = null;
       }

       connect() {
         // Opens EventSource to /api/deploy/stream
         // This receives NARRATION events (metadata) from the backend
         // The actual deployment is running separately in Claude Code
         this.source = new EventSource(this.url);

         this.source.addEventListener('start', (e) => { ... });
         this.source.addEventListener('table', (e) => { ... });
         this.source.addEventListener('relationship', (e) => { ... });
         this.source.addEventListener('complete', (e) => { ... });
         this.source.addEventListener('error', (e) => { ... });

         this.source.onerror = (err) => {
           console.error('SSE error:', err);
           // EventSource auto-reconnects unless explicitly closed
         };
       }

       close() {
         if (this.source) this.source.close();
       }
     }
     ```
   - Parse JSON data from each event
   - Handle reconnection gracefully
